## WeWrite 图片规则

每个观点段落都用 Kie 生图（`kie-image-generator` skill）生成配图来增强/代替文字，配图和内容一一对应，不放纯文字观点。

## Image Hosting Lessons

GitHub图床：https://github.com/JasonXieJ01/imgbed
- 每次上传图片时，必须在仓库根目录新建**日期命名文件夹**（如 `2026-04-23/`）
- 文件夹内按类型命名：`cover_*.png`、`img1.png`、`img2.png`、`img3.png`
- 用 `git add → git commit → git push` 上传，不使用 GitHub API
- 图片上传完成后，Notion 页面使用 `raw.githubusercontent.com` 链接引用
